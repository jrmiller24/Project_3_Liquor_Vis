Interactive Maps Plotting Liquor License Types in Denver County

The Final 3 Group (#RIP Final 4 Group) dove back into the newly minted liquor license data following the legislative approval of bill 17-1370 on Jan. 1, 2019, which legalizes sales of full strength beer at retail stores in the state of CO. We have created three views to highlight this data, broken down by separate neighborhoods within Denver County. The cluster map is an interactive map that can be filtered by license type. The bar chart provides site users with a count of different liquor license types. The pie chart uses a javascript library called granim.js to highlight the visual well-being of the site.
